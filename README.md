# Kyron

Moteur de recherche documentaire local
Permet l'indexation et la recherche plein texte rapide dans des collections de documents PDF 

## Fonctionnalités

* **Indexation :** Extraction de texte via la bibliothèque `PyMuPDF`.
* **Recherche :** Utilisation du module FTS5 (Full-Text Search) de SQLite.

## Prérequis

* Python 3.XX
* Bibliothèques listées dans `requirements.txt`
